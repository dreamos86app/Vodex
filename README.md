# Vodex Platform

AI-native platform for building and publishing software — [vodex.dev](https://vodex.dev).

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Verification

```bash
npm run typecheck
npm run build
npm run verify:brand-migration
npm run verify:paddle-domain-review
```
